# AgentMoveTurnCommand

このMakeCode for Minecraftプロジェクトは、チャットコマンド「w」が入力されたときに、エージェントが前進し、向きを変える動作を行います。

## 表示されるブロック

- チャットコマンド入力時
- エージェントを動かす
- エージェントの向きを変える

## カスタマイズ方法

`main.ts`または`main.blocks`を編集することで、以下のような挙動に変更可能：

- 移動距離の変更
- 回転方向の変更
- 複数の動作の組み合わせ

## 使い方

1. MakeCodeエディタでプロジェクトを読み込む
2. チャットで「w」と入力
3. エージェントが前進し、左に向きを変える
