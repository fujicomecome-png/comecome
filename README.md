# AgentCommandOnly

このMakeCode for Minecraftプロジェクトは、チャットコマンド「w」が入力されたときに何も処理を行いません。

## 表示されるブロック

- チャットコマンド入力時
- エージェントを動かす
- エージェントの向きを変える

## カスタマイズ方法

`main.ts`または`main.blocks`を編集することで、以下のような挙動に変更可能：

- エージェントを前進させる
- エージェントの向きを左右に変える
- 複数の動作を組み合わせる

## 使い方

1. MakeCodeエディタでプロジェクトを読み込む
2. チャットで「w」と入力
3. 処理は実行されません（空のテンプレート）
