# MoveTurnCommand

このMakeCode for Minecraftプロジェクトは、チャットコマンド「w」が入力されたときに、プレイヤーが移動・向き変更を行う動作を定義します。

## 表示されるブロック

- チャットコマンド入力時
- プレイヤーを動かす
- プレイヤーの向きを変える

## カスタマイズ方法

`main.ts`または`main.blocks`を編集することで、以下のような挙動に変更可能：

- 前進のみ
- 左右に向きを変えるのみ
- 移動と回転の順序変更
- 移動距離や回転方向の変更

## 使い方

1. MakeCodeエディタでプロジェクトを読み込む
2. チャットで「w」と入力
3. 定義された動作が実行される
